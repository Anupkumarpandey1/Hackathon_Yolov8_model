# SpaceHack Submission

## How to Run

### 1. Environment Setup
- Install dependencies:
  ```
  pip install -r requirements.txt
  ```

### 2. Training
- To train the model:
  ```
  python train.py
  ```

### 3. Prediction
- To run predictions:
  ```
  python predict.py
  ```

### 4. Visualization
- To visualize ground truth bounding boxes:
  ```
  python visualize.py
  ```
  - Use keys:
    - `d` : Next image
    - `a` : Previous image
    - `t` : Switch to train set
    - `v` : Switch to validation set
    - `q` or `Esc` : Quit

### 5. Report
- See `Report.pdf` for methodology, results, and analysis.

---

**3. Add a `requirements.txt` file:**
```
ultralytics
opencv-python
pyyaml
torch
```
